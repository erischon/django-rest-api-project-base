# My Django REST API Base Template

My Django REST API base project.

## Installing
- [ ] git clone https://github.com/erischon/django-rest-api-project-base.git
- [ ] new folder: .venv
- [ ] new file: .env with SECRET_KEY = ...
- [ ] pipenv shell
- [ ] pipenv install --dev
- [ ] python manage.py runserver
